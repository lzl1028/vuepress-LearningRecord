# 页面布局

## 标准文档流(padding + margin + 负margin) + 浮动float + 定位

## 百分比布局(流式布局): px单位 用 %num代替, 占父级元素的百分比

## flex弹性布局: 主轴 辅助轴的几个属性

## grid栅格布局: 使用框架中的类名来替代: 本质上还是百分比布局
